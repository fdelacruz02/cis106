# What is Linux?

Linux is a unix like operating system that is available through various distributions.
Most Linux distros are highly customizable and give large amount of control to the user. 

# 3 reasons to use Linux

* Linux is an operating system that is free. 
* Linux has a large community of users that support its various distributions.
* Linux runs the internet.
